# Catatan Perbaikan

1. **Tombol Cetak Struk Bergambar**: Tombol Cetak/Web di daftar transaksi sudah saya modifikasi! Sekarang, jika Anda menekan tombol **"🖼️ Web Print"** (sebelumnya bernama "Nota Web (HTML)"), sistem akan membuka halaman browser baru yang berisi **gambar nota asli** (sama persis dengan yang dikirim ke Telegram/WhatsApp) dan tombol untuk **Cetak Struk**. Karena ini menggunakan format gambar resolusi tinggi, struk tidak akan blur atau rusak saat di-print.
